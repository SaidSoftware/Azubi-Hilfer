# Azubi-Hilfer

<img width="1917" height="1015" alt="AzubiHilfer" src="https://github.com/user-attachments/assets/78e5f34c-68b5-4ca3-97a3-c0a9eaeffe16" />


**Azubi-Hilfer** is a desktop application built with **C#**, **WPF**, and **.NET 8**.
The app is designed to help apprentices organize learning materials, tasks, and useful information in one place.

---

## 🚀 Features

* Clean desktop interface
* Organized structure with Models, Views, Services, and Data layers
* Built with modern .NET 8
* Local project structure for easy extension
* Designed for apprenticeship and learning support

## ▶️ Getting Started

### Requirements

* Windows
* Visual Studio 2022
* .NET 8 SDK

### Run the project

```bash
git clone https://github.com/SaidSoftware/Azubi-Hilfer.git
cd Azubi-Hilfer
```

Open `AzubiHilfer.sln` in Visual Studio and start the application.
